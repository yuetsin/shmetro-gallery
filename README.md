# shmetro-gallery

上海轨道交通非官方 macOS 客户端。

---

## 功能

- [x] 查看各线路当前运营情况

- [x] 查看 16 条轨道交通线路站点信息

- [x] 查看各站点首末班车时间、无障碍设施信息
